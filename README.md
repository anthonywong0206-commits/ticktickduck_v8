# Tick Tick Duck v8.6 - 戶外活動離隊聲明一頁版排版更新

## 更新內容
- 重新整理「戶外活動離隊聲明」生成文件排版。
- 系統輸入後生成的資料以紅色顯示。
- 申請人姓名、機構名稱、舉行日期、離隊時間、活動名稱會完整顯示。
- 簽名位置保持原有右側簽署區設計。
- PDF / 圖片 / 列印輸出設定為 A4 直向一頁內顯示。
- 保持零 npm 靜態版，毋須 npm install 或 build。

## 使用方法
1. 解壓 ZIP。
2. 將所有檔案覆蓋到你的 GitHub repo。
3. Commit changes。
4. Vercel / GitHub Pages 會自動更新。

## Vercel 設定
- Framework Preset: Other
- Install Command: 留空
- Build Command: 留空
- Output Directory: 留空或 `.`
