# Tick Tick Duck v8.7 - 戶外活動離隊聲明簽名下方一頁版

## 更新內容
- 重新整理「戶外活動離隊聲明」生成文件排版，格式接近上一張示範圖。
- 系統輸入資料以紅字顯示。
- 簽名位置縮細，並列於文件內容下方、表格編號頁尾之上。
- 保留申請人、見證人、負責人三組簽署資料。
- PDF / 圖片 / 列印輸出保持 A4 直向一頁內。
- 零 npm 靜態版，毋須 npm install 或 build。

## 使用方法
1. 解壓 ZIP。
2. 將所有檔案覆蓋到你的 GitHub repo。
3. Commit changes。
4. Vercel / GitHub Pages 會自動更新。

## Vercel 設定
- Framework Preset: Other
- Install Command: 留空
- Build Command: 留空
- Output Directory: 留空或 `.`
